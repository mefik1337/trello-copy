# Trello copy
## How it works?
U can drag collections, items and move them between each other. Also u can swap places the columns.
## Technologies used
* JS
* React.js (hooks)
* SCSS
## How to install?
* git clone ```https://github.com/mefik1337/trello-copy.git```
* ```npm install```
* ```npm start```
* ```http://localhost:3000/``` to see a preview.
